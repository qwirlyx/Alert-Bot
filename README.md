# HinkalychAlertBot

HinkalychAlertBot — сервис мониторинга сайтов и внутренних веб-сервисов с уведомлениями в Telegram.

Проект проверяет доступность HTTP-страниц и TCP-портов, сохраняет состояние проверок и показывает статус сервисов в простой веб-панели. При падении сервиса бот отправляет уведомление в Telegram, а после восстановления сообщает, что сервис снова работает.

## Возможности

- проверка HTTP-сервисов по статусу ответа;
- проверка TCP-портов;
- контроль количества неудачных попыток перед отправкой уведомления;
- уведомления о падении и восстановлении сервиса;
- панель состояния с последней проверкой и ошибкой;
- запуск проверок через cron;
- ведение логов работы мониторинга.

## Из чего состоит сервис

- `index.php` — веб-панель состояния сервисов;
- `monitor.php` — основной скрипт проверки доступности;
- `lib/Telegram.php` — отправка уведомлений в Telegram;
- `lib/Logger.php` — запись логов;
- `config.php` — базовая конфигурация проекта;
- `config.local.example.php` — пример локальной конфигурации;
- `data/` — состояние проверок и служебные файлы.

## Технологии

PHP, cURL, Telegram Bot API, cron, JSON-хранилище состояния.
